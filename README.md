# wsl

wsl isntallieren
```powershell
wsl --install
```

wsl updaten
```powershell
wsl --update
```

wsl Linux versionen anzeigen
```powershell
wsl --list --online
```

wsl Linux Version installieren
```powershell
wsl --install -d <DistroName>
```
